# Руководство по тестированию приложения Adventy

## Для HR-специалистов

Это руководство поможет вам протестировать приложение Adventy в рабочей среде (production).

---

## Что такое приложение Adventy?

Adventy - это календарь приключений, где каждый день с 10 декабря 2025 по 31 декабря 2025 года представлен карточкой. Пользователи могут открывать карточку текущего дня, чтобы увидеть задание на сегодня.

---

## Как открыть приложение?

1. Откройте браузер (Chrome, Firefox, Edge и т.д.)
2. Перейдите по адресу вашего приложения (например: `https://adventy.yourcompany.com`)
3. Вы увидите календарь с карточками дат

---

## Обычное использование

### Просмотр задания на сегодня

1. Найдите карточку с надписью **"Today"** (сегодняшняя дата выделена фиолетовым цветом)
2. Нажмите на эту карточку
3. Карточка откроется и покажет задание на сегодня
4. Чтобы закрыть карточку, нажмите на крестик (×) в правом верхнем углу или нажмите клавишу **Escape**

### Просмотр других дат

- Если вы попытаетесь открыть карточку прошедшей даты, вы увидите сообщение: **"Дата уже прошла, пора выполнять следующие задания"**
- Если вы попытаетесь открыть карточку будущей даты, вы увидите сообщение: **"Обожди, не торопись, скоро новое приключение будет доступно!"**

## Тестирование с помощью специальных параметров

Для тестирования приложения вы можете использовать специальные параметры, которые позволяют просматривать задания для любых дат, включая прошедшие и будущие.

### Как использовать параметры?

Добавьте параметры в конец адреса сайта после знака `?`.

**Формат:**
```
https://adventy.yourcompany.com?first=СЕКРЕТ1&second=СЕКРЕТ2&third=СЕКРЕТ3&showPast=true
```

### Параметры для отображения дат

**Параметр `showPast`:**
- По умолчанию прошедшие даты **скрыты** из календаря
- Чтобы показать прошедшие даты, добавьте `showPast=true` в адрес:
  ```
  https://adventy.yourcompany.com?showPast=true
  ```
- Это позволит увидеть все даты, включая прошедшие, в календаре

### Секретные значения для обхода валидации

Для тестирования используйте следующие секретные значения:

1. **Первый параметр (first)** - позволяет просматривать даты вне диапазона:
   ```
   CggAEEUYFhgeGDkyCggBEAAYgA_secret_to_skip_date_in_range
   ```

2. **Второй параметр (second)** - позволяет просматривать будущие даты:
   ```
   EgZjaHJvbWUyCggAE_secret_to_skip_date_has_not_appeared
   ```

3. **Третий параметр (third)** - позволяет просматривать прошедшие даты:
   ```
   AYgAQYogQyCggDEAAYogQYiQUyC_secret_to_skip_date_passed
   ```

## Сценарии тестирования

### Сценарий 1: Просмотр задания для прошедшей даты

**Цель:** Проверить, что можно открыть задание для даты, которая уже прошла.

**Шаги:**
1. Откройте браузер
2. Введите адрес с параметрами `showPast=true` и `third`:
   ```
   https://adventy.yourcompany.com?showPast=true&third=AYgAQYogQyCggDEAAYogQYiQUyC_secret_to_skip_date_passed
   ```
   - `showPast=true` - показывает прошедшие даты в календаре
   - `third` - позволяет открыть задание для прошедшей даты
3. Нажмите Enter
4. Найдите любую карточку с прошедшей датой (дата будет серого/синего цвета)
5. Нажмите на карточку
6. Карточка должна открыться и показать задание для этой даты

**Ожидаемый результат:**
- Прошедшие даты отображаются в календаре
- Карточка открывается и показывает задание, даже если дата уже прошла

---

### Сценарий 2: Просмотр задания для будущей даты

**Цель:** Проверить, что можно открыть задание для даты, которая еще не наступила.

**Шаги:**
1. Откройте браузер
2. Введите адрес с параметром `second`:
   ```
   https://adventy.yourcompany.com?second=EgZjaHJvbWUyCggAE_secret_to_skip_date_has_not_appeared
   ```
3. Нажмите Enter
4. Найдите любую карточку с будущей датой
5. Нажмите на карточку
6. Карточка должна открыться и показать задание для этой даты

**Ожидаемый результат:** Карточка открывается и показывает задание, даже если дата еще не наступила.

---

### Сценарий 3: Просмотр задания для даты вне диапазона

**Цель:** Проверить работу с датами, которые находятся вне диапазона 10-31 декабря 2025.

**Шаги:**
1. Откройте браузер
2. Введите адрес с параметром `first`:
   ```
   https://adventy.yourcompany.com?first=CggAEEUYFhgeGDkyCggBEAAYgA_secret_to_skip_date_in_range
   ```
3. Нажмите Enter
4. Попробуйте открыть любую карточку

**Ожидаемый результат:** Приложение работает нормально, даже если дата находится вне обычного диапазона.

---

### Сценарий 4: Комбинированное тестирование

**Цель:** Использовать несколько параметров одновременно для полного тестирования.

**Шаги:**
1. Откройте браузер
2. Введите адрес со всеми параметрами:
   ```
   https://adventy.yourcompany.com?showPast=true&first=CggAEEUYFhgeGDkyCggBEAAYgA_secret_to_skip_date_in_range&second=EgZjaHJvbWUyCggAE_secret_to_skip_date_has_not_appeared&third=AYgAQYogQyCggDEAAYogQYiQUyC_secret_to_skip_date_passed
   ```
3. Нажмите Enter
4. Теперь вы можете:
   - Видеть все даты, включая прошедшие (благодаря `showPast=true`)
   - Открывать карточки любых дат без ограничений (благодаря секретным параметрам)

**Ожидаемый результат:**
- Все даты отображаются в календаре
- Все карточки открываются и показывают задания, независимо от даты

## Важные замечания

### Безопасность

⚠️ **ВАЖНО:** Эти параметры предназначены только для тестирования! Не делитесь этими секретными значениями с обычными пользователями.

### Когда использовать параметры?

Используйте параметры только когда нужно:
- Проверить задания для прошедших дат (используйте `showPast=true` чтобы увидеть их в календаре)
- Проверить задания для будущих дат
- Протестировать работу приложения в различных сценариях

### Обычные пользователи

Обычные пользователи должны использовать приложение **без параметров**. Они:
- Увидят только сегодняшнюю и будущие даты (прошедшие даты скрыты)
- Смогут открывать только карточку текущего дня

## Часто задаваемые вопросы

### Вопрос: Что делать, если карточка не открывается?

**Ответ:**
1. Убедитесь, что вы используете правильный секретный код
2. Проверьте, что адрес введен правильно (без пробелов)
3. Попробуйте обновить страницу (F5 или Ctrl+R)

### Вопрос: Можно ли использовать параметры на мобильном телефоне?

**Ответ:** Да, параметры работают на всех устройствах. Просто введите полный адрес с параметрами в адресной строке браузера.

### Вопрос: Нужно ли вводить все три параметра?

**Ответ:** Нет, вы можете использовать только те параметры, которые нужны для вашего теста. Например, если вы хотите проверить только прошедшие даты, используйте только параметр `third`.

### Вопрос: Что делать, если я вижу ошибку?

**Ответ:**
1. Проверьте правильность секретных кодов
2. Убедитесь, что дата существует в системе
3. Свяжитесь с технической поддержкой, если проблема сохраняется
